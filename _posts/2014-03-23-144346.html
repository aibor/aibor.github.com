--- layout: article categories: articles date: 2014-03-23 14:52:50 title: Passwort aus git Repo history entfernen --- Mit folgendem Befehel kann man die git history neu schreiben lassen, um z.B. Passwörter aus einer tracked datei zu löschen: {% highlight console %} $ git filter-branch --tree-filter "sed -i 's///' pfad/zur/datei" {% endhighlight %} Statt dem exakten Passwort sollte man einen passenden regex nutzen oder anderweitig vermeiden, dass dieses anschließend in der bash history auftaucht.
